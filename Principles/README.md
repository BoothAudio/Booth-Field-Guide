# Principles

Purpose

This folder contains individual Booth principles.

Unlike the Constitution, principles are atomic.

Each principle should describe exactly one idea.

Every principle should be applicable across multiple disciplines.

Engineering.

Design.

Product.

Marketing.

Customer support.

Leadership.

If a principle only applies to one specific situation, it probably belongs in Decisions instead.

---

## Lifecycle

Observation

↓

Repeated Decisions

↓

Candidate Principle

↓

Official Principle

↓

Possible Constitutional Principle

---

A principle should survive multiple unrelated decisions before being considered official.
