# Open Question: Field Guide Governance Details

Status: Open
Opened: 2026-07-22

## Question

How formal should Booth Field Guide review, approval, and publication become as
the company and contributor base grows?

## Why It Matters

The guide should reduce founder cognitive load without letting candidate ideas
become established Booth philosophy without approval.

## Current Evidence

Nick has approved the Field Guide as the canonical source for Booth philosophy
and institutional reasoning. Nick has also defined which changes the agent may
draft autonomously and which require founder approval.

## Possible Directions

- Keep a lightweight Git workflow while Booth is founder-led.
- Add explicit review states if more people begin contributing decisions.
