# Open Questions

Purpose

This folder contains ideas that feel important but have not yet earned a permanent place in Booth's philosophy.

An unanswered question is not a weakness.

It is evidence that Booth is still learning.

Questions should remain here until repeated experience provides an answer.

---

Examples

Should Booth ever infer metadata?

What responsibilities does Booth have when AI is uncertain?

How opinionated should recommendations become?

What makes hardware feel like Booth?

How should Booth balance openness with quality?

---

Questions should never be removed simply because they are old.

They should only be removed when Booth has earned an answer.
